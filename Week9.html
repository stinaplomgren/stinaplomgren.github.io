Week 9: Input Devices

Temperature and Humidity

The Basic Idea

This week's assignment was pretty straightforward and seemed easy. We had to connect a sensor of some sort to our Arduino and program it to collect data. I selected a temperature and humidity sensor because I thought it might be helpful for me to monitor the environment for some plants that need higher levels of humitity and heat, like begonias or calatheas.

Programming

This is where I ran into some trouble. Either the code that came with the sensor was faulty or something else was going on but I could not get the code to verify and upload to my Arduino. I looked into many solultions on the internet and asked a TA for help. We ended up spending an entire class period troubleshooting the code and eventually just discarded the code that came with the sensor in favor of code that we found on the Arduino website. It ended up working, but then I ran into another problem where the schematic that the sensor came with was incorrect so I had to find another way to wire the sensor to the Arduino. I was super worried that I would fry the sensor or my Arduino but I figured out how to make it work.
